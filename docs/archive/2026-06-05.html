Agentic Monte Carlo controls black-box LLM agents without fine-tuning

Most production stacks call Claude or Oracle Fusion APIs with no weight access, so RL is off the table. AMC treats the API agent as a prior and samples from the optimal policy via Sequential Monte Carlo.
